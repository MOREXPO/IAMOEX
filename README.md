## 🛠 Instalación y ejecución local

```bash
# Clonar el repositorio
git clone https://github.com/MOREXPO/IAMOEX.git

# Acceder al directorio
cd IAMOEX

# Instalar dependencias
npm install

# Ejecutar la app en modo desarrollo
npm start
